## Overview

HyBridger was built to show how a professional-grade, production-ready web architecture comes together using `Next.js`, `Contentful`, `TailwindCSS`, and `TypeScript`. The project demonstrates real-world content modeling, `GraphQL` querying patterns, image optimization, and scalable UI composition, offering a practical example of how engineering and design systems integrate in modern workflows.

## Architecture
HyBridger uses the `Next.js App Router` with a well-organized structure:

<ul className="list">
<li>Server components for headless CMS data fetching</li>
<li>Client components for interactive UI pieces</li>
<li>Centralized `GraphQL` utilities</li>
<li>Custom responsive image component</li>
<li>Typed data models for every query and CMS field</li>
<li>ISR revalidation through a custom API route</li>
</ul>
This ensures predictable data flow and fast render times.
