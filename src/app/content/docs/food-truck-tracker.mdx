## Overview

Food Truck Tracker is a web app built around real-time scheduling. It balances real-time responsiveness with rendering and runtime constraints in a hybrid SSR/CSR environment, efficient state management, and mobile-first performance. The result is a vibrant, user-friendly interface that reduces friction and keeps interactions fast and effortless.

## Architecture

**UI & State Management:**

Optimized state updates by `useEffect()` to minimize unnecessary re-renders, ensuring smooth interactions under frequent time-based updates.

**SSR/CSR Boundary Handling:**

Resolved hydration issues by isolating time-sensitive logic to the client, preventing server-client render mismatches in production.

**URL State Management with useSearchParams:**

Enabled dynamic rendering by leveraging `Next.js` `App Router` `useSearchParams` to persist query parameters across navigation, ensuring consistent routing behavior in edge cases while preserving scroll position.

## Features

**1. Real-Time Scheduling:**

Built a live clock to compute durations from the current client time, accurately calculating upcoming arrivals down to the millisecond and formatting output in 12-hour time.

**2. Swiper Carousel Integration:**

Implemented a touch-optimized carousel using `Swiper.js` with autoplay, maintaining responsive layouts and consistent slide dimensions across viewports.

**3. Hot Menu Item Search:**

Develop a client-side search with controlled input state and dynamic filtering logic to return relevant menu items with minimal re-rendering.

**4. Schedule Sorting:**

Implement accessible sorting controls on the schedule table header to allow users to toggle between ascending and descending order.

**5. Food Truck Modal:**

Designed and built a lightweight modal to display detailed food truck info.
